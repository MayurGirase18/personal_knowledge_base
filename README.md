# Personal Knowledge Base

A command-line Notes Management System built with Python that stores notes in a JSON file. The project demonstrates file handling, CRUD operations, modular programming, and structured data storage.

## Features

- Create Note
- View Notes
- Edit Note
- Delete Note
- Search Notes
- JSON File Storage

## Tech Stack

- Python 3
- JSON
- UV

## Project Structure

```text
src/
data/
tests/
```

## Run

```bash
uv run python src/main.py
```
